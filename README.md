This project is a classic Snake Game built to understand core game development concepts using modern frontend tools.
It focuses on grid-based movement, game loops, and real-time state updates in a React environment.

The game is implemented using a 2D grid system, where the snake moves cell-by-cell, grows after eating food, and ends when it collides with walls or itself.
